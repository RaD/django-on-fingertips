#################
Введение в Django
#################

Django представляет собой хорошее приложение на языке Python, которое позволяет
ударными темпами разрабатывать WEB проекты. Но в этом, широко рекламируемом
утверждении есть одна небольшая хитрость. Это всё абсолютная правда, если у вас
есть достаточной опыт в языке Python и вы умеете проектировать, неважно что:
база данных, наборы компонентов и так далее. В других случаях всё будет немного
медленнее, но всё равно интересно.

Разработка WEB проекта является многогранной задачей, в которой могут
участвовать идеологи, проектировщики, дизайнеры, разработчики и ещё какие-то
неизвестные люди. Django не избавит вас от решения их задач, но сможет
существенно их облегчить. Об этом будет подробно рассказано в последующих
главах.
